# **App Name**: SignAI Web

## Core Features:

- Landing Page - Hero Section: Hero section with a title, subtitle, two CTAs (Request pilot, See demo), and an image of the wearable in use.
- Informational Pages: Pages detailing 'How it works', 'Sectors', and 'Use Cases', including diagrams and benefit explanations tailored to each section.
- Contact Form with HubSpot Integration: Contact form designed for lead capture, with fields that can be mapped to HubSpot for marketing automation and CRM integration.
- Blog: A blog section to provide informative articles about SignAI's technology, applications, and impact, enhancing SEO and user engagement.
- Accessibility Compliance: Implementation of WCAG guidelines for accessibility, ensuring the website is usable for people with disabilities.
- ML powered text generation: Use of an LLM to generate human sounding responses on the terms and conditions page. This feature includes a tool for reviewing and editing proposed responses to comply with regulations.

## Style Guidelines:

- Primary color: Technology Blue (#176BFF), same as the original landing page, for a modern and reliable feel.
- Background color: Very light blue (#EBF2FF), creating a clean and spacious layout.
- Accent color: A slightly darker blue (#3B82F6) to highlight CTAs and interactive elements.
- Font: 'Inter', a legible and modern sans-serif, for both headings and body text to ensure clear and accessible readability. Note: currently only Google Fonts are supported.
- Modular design with distinct sections, ample whitespace, and well-organized cards/blocks to display information effectively.
- Use clear and professional icons to visually represent sectors, benefits, and features, enhancing the overall user experience.
- Subtle transitions and animations to enhance interactivity and user engagement, ensuring a smooth browsing experience.
